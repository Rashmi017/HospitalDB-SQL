# HospitalDB — Complete SQL Project

## About
A complete MySQL hospital database project.

## Tables
- Patients (200 records)
- Labs (200 records)
- Diagnoses (10 records)
- Outcomes (5 records)

## How to Run
1. Open MySQL Workbench
2. File → Open SQL Script → select HospitalDB.sql
3. Click ⚡ Run

## Skills Used
- MySQL 8.0
- Joins, Aggregations, Subqueries
- Database Design & Foreign Keys`
