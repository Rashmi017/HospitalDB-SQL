# HospitalDB — Complete SQL Project

A full MySQL hospital database with:
- 4 tables: Patients, Labs, Diagnoses, Outcomes
- 200 patient records & 200 lab records
- 8 analytical queries (joins, aggregations, trends)

## How to Run
1. Open MySQL Workbench
2. Run `HospitalDB.sql`
3. Execute queries from `queries/analytical_queries.sql`
